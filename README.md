# mycycle
mycycle is a fully self-hosted personal period tracking web application built and designed to run on a linux server (Ubuntu v24.04.4). It runs entirely on hardware the user owns, meaning the user's health data never goes to a third-party company like Apple, Google, Flo or Clue and there's no subscription fee. The app can be accessed from any device (phone, Mac, Windows), both inside and outside the network to which the linux server connects by using a personal domain. One important trade off compared to a cloud service is that the user is responsible for keeping it running and keeping the data secure.

## Features
- Track which days the period occurred by clicking on a calendar
- Log symptoms for any day (cramps, headache, bloating, etc.) with optional notes
- Track the IUD insertion date and see when it needs replacement
- Uses a statistical algorithm to predicts the next 3 periods based on historical data
- See the current cycle phase (menstrual, follicular, ovulation, luteal) with tips
- See the fertile window and predicted ovulation day
- Sync data across all the devices through your home server
